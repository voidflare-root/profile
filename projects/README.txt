Projects folder guide

Yahan apne project files upload karo. Website Projects tab me files auto show karegi.

Direct file upload:
example.zip
my-app.apk
source-code.zip

Link project add karna ho to .txt file banao:

name: My App
description: Android app source code and APK download.
download: https://example.com/my-app.zip

Website par user ko project card, download button aur share button dikhega.
README file website me show nahi hogi.
