Projects folder guide

Yahan apne project download links add karo. Website Projects tab me auto cards banayegi.

Simple format:

app https://example.com/my-app.apk
src https://example.com/source-code.zip
zip https://example.com/extra-files.zip

Rules:
- Ek line = ek project download card.
- File name se card ka base name banega.
- app/apk line App card banayegi.
- src/source/code line Source Code card banayegi.
- zip line Zip card banayegi.
- Download button wahi link direct open/download karega.

Example agar file ka naam MyTool.txt hai:

app https://example.com/mytool.apk
src https://example.com/mytool-source.zip

README file website me show nahi hogi.
